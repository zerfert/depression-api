Acerca del Proyecto

Información sobre el sistema de detección de depresión estudiantil

Objetivo del Proyecto

Este sistema utiliza técnicas de aprendizaje automático para evaluar el riesgo de depresión en estudiantes universitarios. El objetivo es proporcionar una herramienta de detección temprana que pueda ayudar a identificar estudiantes que podrían beneficiarse de apoyo profesional en salud mental.

Metodología

El modelo predictivo fue entrenado utilizando un dataset de estudiantes universitarios que incluye variables demográficas, académicas, de estilo de vida y de salud mental. Se emplearon técnicas de:

  • Selección de características con Random Forest
  • Regresión logística para la predicción
  • Validación cruzada para evaluar el rendimiento
  • Normalización de datos numéricos

Características del Modelo

Dataset

Datos reales de estudiantes universitarios con múltiples variables de riesgo

IA/ML

Algoritmos de machine learning optimizados para detección de patrones

Validado

Modelo validado con métricas de precisión, recall y F1-score

Limitaciones y Consideraciones

Importante - Esta NO es una herramienta de diagnóstico médico

  • Los resultados son indicativos y no constituyen un diagnóstico profesional
  • La herramienta está diseñada para fines educativos y de investigación
  • Siempre consulte a un profesional de salud mental calificado
  • En caso de crisis o pensamientos suicidas, busque ayuda inmediata
